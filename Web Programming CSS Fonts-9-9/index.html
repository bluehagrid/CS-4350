CSS Measurement Units

  • %: Defines a measurement as a percentage
  • cm: Defines a measurement in cenitmeters
  • in: Defines a measurement in inches
  • mm: Defines a measurement in millimeters
  • px: Defines a measurement in pixels
  • pt: Defines a measurement in points. A point is 1/72 of an inch
  • em: A relevant measurement for font hight. Each "em" unit is 12 pt.
  • ex: Relative to a font's 'x' height
  • pc: in picas - a pica = 12 points
  • vw: viewport width, which stands for a percentage of the browser's width
  • vh: viewport height, which stands for a percentage of the browser's height
